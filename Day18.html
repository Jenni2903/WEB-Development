CSS width vs max-width

Width is a fixed value that sets the width of an element to a specific size, regardless of the size of its parent container or the size of the content within it. For example, if you set the width of an element to 500px, it will always be 500px wide. Max-width, on the other hand, sets the maximum width of an element.
